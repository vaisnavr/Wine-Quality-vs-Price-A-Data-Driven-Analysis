# 🍷 Wine Quality vs Price: A Data-Driven Analysis

## 📊 Project Presentation
📄 [View Full Presentation](wine-price-analysis-presentation.pdf)

## 🔍 Overview
This project explores whether higher-priced wines are actually better rated by consumers. Using real-world data from Vivino, we analyze the relationship between wine price and ratings through statistical analysis and visualization.

## 🎯 Research Question
Do expensive wines have higher ratings?

## 📈 Key Findings
- Weak positive relationship between price and rating
- R² = 0.233 → price explains only ~23% of variation
- Price is NOT a strong predictor of quality
- Ratings are similar across wine types

## 🧠 Conclusion
Higher price does not necessarily mean better wine.
